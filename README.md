# 🎯 AI任务规格化工具

> 用户以日常口语描述任务 → 多轮澄清关键信息 → 自动生成专业任务规格书

本工具是一个前端Web应用，通过与兼容OpenAI API格式的大语言模型（如DeepSeek、通义千问、GPT-4o-mini等）交互，
将模糊、口语化的任务需求逐步转化为结构清晰、边界明确的专业规格书，
并支持一键复制为**锚点+规格书格式**，可直接投喂给下游智能体用于规划与执行。

---

## ✨ 主要特性

- **口语输入，智能引导**：只需用日常语言描述想做的事，AI会主动追问关键缺失信息（路径、文件类型、交付物等）。
- **多轮对话式澄清**：支持自然交互，可在对话中补充或修改任务细节，直至规格书定稿。
- **专业规格书生成**：最终产出包含**目标、范围、路径与文件约束、交付物、注意事项**等章节的规格文档。
- **任务类型自动推断**：通过`routing_hint`标记任务大类（文件目录操作 / Excel批量测试用例 / 表结构分析 / 联网搜索 / 通用），便于下游流程路由。
- **双格式一键复制**：
  - **文本格式**：清晰的人类可读规格书 + 原始指令锚点。
  - **锚点格式**：`【任务规格化·原始锚点】` + 用户原话 + `【任务规格化·已定稿规格书】` + 规格书正文，可直接粘贴给支持锚点优先的智能体。
- **本地存储配置**：API地址、密钥、模型名称、系统提示词均保存在浏览器`localStorage`中，无需重复填写。
- **视觉友好**：左右双栏布局（聊天区 + 实时规格书面板），消息气泡、加载动画、深色/浅色自适应界面。

## 🖥️ 界面预览

| 区域 | 功能说明 |
|------|----------|
| 左侧聊天区 | 用户与AI的对话记录，支持口语描述、确认、修改等交互。 |
| 右侧规格书面板 | 实时展示当前待确认问题，定稿后显示完整任务规格书与原始指令锚点。 |

| 底部输入框 | 键入任务描述或补充信息，支持 <kbd>Ctrl+Enter</kbd> 发送。 |
